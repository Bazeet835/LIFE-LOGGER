### Step 3 â€” Create a Private GitHub Repository

1. Go to [GitHub](https://github.com/) and create a new repository.
2. Name it appropriately (e.g., `life-data`).
3. Set the repository to **Private**.

### Step 4 â€” Configure Environment Variables

Create a `.env` file in the root directory with the following content:

```env
TELEGRAM_TOKEN=your_telegram_bot_token
GITHUB_TOKEN=your_github_personal_access_token
GITHUB_REPO=your_username/your_repository_name
GITHUB_BRANCH=main
OWNER_ID=your_telegram_user_id
```

**Note:** Replace the placeholders with your actual values.

---

### Step 5 â€” Generate a GitHub Personal Access Token

1. Navigate to [GitHub Settings](https://github.com/settings/tokens).
2. Click on **"Generate new token"**.
3. Select the following scopes:
   - `repo` (Full control of private repositories)
4. Generate the token and copy it. **Store it securely.**
